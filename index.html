La conexión con Klaviyo no funciona desde tu ordenador — el navegador lo bloquea por seguridad (se llama CORS).
Para activarlo, sube este archivo a GitHub Pages (gratis, 3 minutos):

1 → Ve a github.com y crea una cuenta gratuita
2 → Crea un repositorio nuevo (cualquier nombre)
3 → Sube este archivo (que se llama index.html)
4 → Ve a Settings → Pages → Branch: main → Save
5 → Copia la URL que te da y ábrela — Klaviyo funcionará

Sync Google Sheets

Configura una vez (5 min) y tendrás tu CRM en Google Drive.

Flujo: Exporta CSV desde aquí → en tu Sheet ve a 🔴 MJ Hub → Importar CSV → listo con formato automático.
1
Crea carpeta en Google Drive
Nueva carpeta → "MJ Hub — Clientes" → subcarpetas: CRM, Propuestas, Contratos.
2
Crea Google Sheet nueva
Dentro de la carpeta CRM, crea la hoja "Pipeline de Leads".
3
Pega el Apps Script
En tu Sheet: Extensiones → Apps Script → pega el código → guarda.
function onOpen(){SpreadsheetApp.getUi().createMenu('🔴 MJ Hub').addItem('Importar CSV','importarCSV').addItem('Formatear','formatear').addToUi();} function importarCSV(){var ui=SpreadsheetApp.getUi();var r=ui.prompt('Pega el CSV del Hub:',ui.ButtonSet.OK_CANCEL);if(r.getSelectedButton()!==ui.Button.OK)return;var csv=r.getResponseText().trim();if(!csv){ui.alert('Vacío');return;}var rows=csv.split('\n').map(function(r){return r.split(',').map(function(c){return c.replace(/^"|"$/g,'');});});var s=SpreadsheetApp.getActiveSpreadsheet().getActiveSheet();s.clearContents();s.getRange(1,1,rows.length,rows[0].length).setValues(rows);formatear();ui.alert('✅ '+( rows.length-1)+' leads importados.');} function formatear(){var s=SpreadsheetApp.getActiveSpreadsheet().getActiveSheet();var r=s.getLastRow();var c=s.getLastColumn();if(r<1)return;s.getRange(1,1,1,c).setBackground('#C8392B').setFontColor('white').setFontWeight('bold');for(var i=2;i<=r;i++)s.getRange(i,1,1,c).setBackground(i%2===0?'#FFF5F3':'#FFFFFF');for(var j=1;j<=c;j++)s.autoResizeColumn(j);s.setFrozenRows(1);}
4
Exporta y sincroniza
Pulsa "Exportar CSV" → en tu Sheet: 🔴 MJ Hub → Importar CSV → pega.
